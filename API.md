## API Structure

### User
- email: `string`
- password: `string`
- name (first + last): `string`
- address: `string`
- firebase id: `int`
- isDoctor: `boolean`
- skills: `array` of strings
- conditions: `array` of strings or `string`?

### Meeting
- members: `array` of ids or emails?
- time: `string`
- address: `string` (address of doctor's office)
